# Projectes Xarop

Aquest repositori conté diversos projectes desenvolupats per Xarop. Consulta l'índex a l'arrel per accedir als projectes publicats a GitHub Pages.

## Publicació a GitHub Pages

1. Ves a la configuració del repositori a GitHub.
2. A l'apartat **Pages**, selecciona la branca `master` i la carpeta `/` (arrel) com a font.
3. Desa i obre l'enllaç que et proporciona GitHub Pages.

# Jocs clàssics minimalistes per a mòbil

Col·lecció de 5 jocs clàssics, estil minimalista, jugables arrossegant amb el dit o el ratolí. Sense botons, sense ombres, sense dissenys complicats. Inspirats en els originals.

## Accés ràpid

- [Arkanoid](arkanoid/index.html): Arrossega la pala per trencar maons
- [Space Invaders](space-invaders/index.html): Arrossega la nau per disparar als invasors
- [Pong](pong/index.html): Arrossega la pala per retornar la pilota
- [Breakout](breakout/index.html): Trenca tots els maons amb la pala
- [Tetris](tetris/index.html): Arrossega per moure peces, encaixa-les!

## Publicació a GitHub Pages
1. Ves a la configuració del repositori a GitHub.
2. A l'apartat **Pages**, selecciona la branca `master` i la carpeta `/` (arrel) com a font.
3. Desa i obre l'enllaç que et proporciona GitHub Pages.

# Arkanoid per a mòbil

Joc Arkanoid fet amb HTML5, CSS i JavaScript, optimitzat per a dispositius mòbils amb controls tàctils.

## Com jugar
- Mou la pala amb els botons tàctils de l'esquerra i la dreta.
- Trenca tots els maons per guanyar!

## Publicació a GitHub Pages

1. Puja la carpeta `arkanoid` al teu repositori de GitHub.
2. Ves a la configuració del repositori a GitHub.
3. A l'apartat **Pages**, selecciona la carpeta `arkanoid` com a font.
4. Desa i obre l'enllaç que et proporciona GitHub Pages.

El joc estarà disponible per jugar des del navegador del mòbil!
